# Java 撰寫方式



## 步驟 1\_撰寫 Java 程式

在記事本內，撰寫下述程式碼，並儲存副檔名為『HelloWorld.java』的檔案

```java
// 撰寫 Hello World 的 Java 程式
public class HelloWorld{

       public static void main(String[] args){

              System.out.println("Hello World!!");

       } // End of main()

} // End of class HelloWorld
```
