# SQL ... END SQL

## 語法

```sql
SQL
   SQL Statement
END SQL
```

{% hint style="info" %}
說明：

表示此 SQL Statement 只單純作為 SQL 語法使用。\
可替代 PREPARE 敘述。\
若須帶入程式內的變數，則須在該語法內的變數前，搭配上 $ 符號。
{% endhint %}
