# MESSAGE

## 語法1

```inform7
MESSAGE "str1" [, "str2", ... ] [ Atrribute ]
```

## 語法 2

```inform7
MESSAGE variable1 [, variable2, ... ] [ Atrribute ]
```

## 語法 3

```inform7
MESSAGE ""
```

{% hint style="info" %}
目的：

顯示訊息於 MESSAGE 行。\
預設位置在 ACTION WINDOW 第二行。
{% endhint %}
