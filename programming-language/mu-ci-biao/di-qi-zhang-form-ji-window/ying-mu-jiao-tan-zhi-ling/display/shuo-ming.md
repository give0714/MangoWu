# 說明

{% hint style="info" %}
目的：

顯示訊息。\
可指定位置。
{% endhint %}

{% hint style="info" %}
備註：

* DISPLAY AT 敘述：\
  會將 display\_list 字串顯示於指定的螢幕位置。\
  row 為螢幕列座標的整數值。\
  column 為螢幕行座標的整數值。
* DISPLAY ARRAY 敘述：\
  會將 programArrayName 陣列資料顯示於螢幕的陣列之中。
* ON KEY 敘述：\
  使用時，一定要使用 END DISPLAY 敘述。
{% endhint %}
