# UPSHIFT( )

## 語法

```
UPDOWNSHIFT( strExpr )
```

{% hint style="info" %}
目的：

將指定英文的字元或字串轉成大寫英文的字元或字串。
{% endhint %}

{% hint style="info" %}
說明：

將 strExpr 中，所有小寫英文字元均轉換成大寫英文字元，若含有非英文字元，則無法轉換。
{% endhint %}
