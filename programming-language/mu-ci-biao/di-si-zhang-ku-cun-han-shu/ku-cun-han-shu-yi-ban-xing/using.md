# USING

## 語法

```
expr1 USING "expr2"
```

{% hint style="info" %}
目的：

將 expr1 的格式，轉換成 expr2 的方式
{% endhint %}

{% hint style="info" %}
格式化字元：

\* ： 將空白轉成 \*。

& ： 將空白轉成 0。

\# ： 不改變空白。

< ： 數值向左調整。

,  ： 數字每 3 位，用 , 隔開。

\- ： 當數值小於 0 時，將會顯示負號。

\+ ： 當數值大於等於 0 時，將會顯示正號，反之顯示負號。

() ： 用於替代負號，也可表示為負數。

$ ： 數值左側填入錢號。

dd ： 表示 01 \~ 31 的天數。

ddd ： 表示星期的縮寫。

mm ： 表示 01 \~ 12 的月份。

mmm ： 表示月份的縮寫。

yy ： 表示 1900 \~ 1999 區間的年份。

yyyy ： 表示 0000 \~ 9999 區間的年份。
{% endhint %}
