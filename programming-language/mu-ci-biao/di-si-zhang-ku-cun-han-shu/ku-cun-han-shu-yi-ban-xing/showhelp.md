# SHOWHELP( )

## 語法

```
SHOWHELP( intExpr )
```

{% hint style="info" %}
目的：

顯示求助訊息。
{% endhint %}

{% hint style="info" %}
說明：

intExpr ： 整數表示式，為求助檔案的求助訊息編號。
{% endhint %}
