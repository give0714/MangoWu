# LENGTH( )

## 語法

```
LENGTH( str )
```

{% hint style="info" %}
目的：

傳回字串的長度。
{% endhint %}

{% hint style="info" %}
說明：

用以統計除尾部空白外的所有位元組資料。

str ：為字串長度或 CHAR 資料型別之變數。
{% endhint %}
