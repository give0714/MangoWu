# COLUMN

## 語法

```
COLUMN intExpr
```

{% hint style="info" %}
目的：

設定後續項目的起始位置。
{% endhint %}

{% hint style="info" %}
說明：

intExpr ： 正整數表示式，若使用於 DISPLAY 敘述，須設定整數值。
{% endhint %}
