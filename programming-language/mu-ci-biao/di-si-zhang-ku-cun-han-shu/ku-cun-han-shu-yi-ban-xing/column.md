# COLUMN

## 語法

```
COLUMN intExpr
```

{% hint style="info" %}
目的：

設定後續項目的起始位置。
{% endhint %}

{% hint style="info" %}
說明：

intExpr ： 正整數表示式，若使用於 DISPLAY 敘述，須設定整數值。
{% endhint %}

{% hint style="danger" %}
前往【 城市芒果留言區 】\
如有建議芒果改進的地方，請前往芒果留言區留言\
使芒果與你們一起成長進步
{% endhint %}
