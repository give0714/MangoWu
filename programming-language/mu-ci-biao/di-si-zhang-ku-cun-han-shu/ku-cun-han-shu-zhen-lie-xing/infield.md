# INFIELD( )

## 語法一

```
INGIELD( )
```

{% hint style="info" %}
目的：

判斷目前欄位名稱是否與輸入值相同。
{% endhint %}

## 語法二

```
INGIELD( screenFieldName )
```

{% hint style="info" %}
目的：

判斷螢幕是否在螢幕欄位中。
{% endhint %}

{% hint style="info" %}
說明：

通常搭配 ONKEY 敘述使用，回傳一布林值。
{% endhint %}
