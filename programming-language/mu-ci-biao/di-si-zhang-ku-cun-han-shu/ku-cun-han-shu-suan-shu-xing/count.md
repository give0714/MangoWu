# COUNT( )

## 語法一

```
COUNT( * )
```

{% hint style="info" %}
目的：

紀錄符合 SQL 條件的資料數量。
{% endhint %}

## 語法二

```
COUNT( [ DISTINCT / UNIQUE ] columnName )
```

{% hint style="info" %}
目的：

紀錄符合 SQL 條件的 columnName 欄位資料數量。
{% endhint %}
