# 資料表命名規則

{% hint style="info" %}
IN dbspaceName：若省略不寫，該資料庫將會建立在 Informix Database Server 的 root dbspace 上。

WITH \[( BUFFERED / LOG MODE ANSI )] LOG ]：若省略不寫，則該資料庫就不會產生 log，後續將無法使用 Transaction 的語法。
{% endhint %}
