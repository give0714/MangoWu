# INT

#### 說明

| 語法   | INT                       |
| ---- | ------------------------- |
| 資料長度 | 4 byte                    |
| 資料範圍 | -2147483647 \~ 2147483647 |

#### 範例

```objectivec
DEFINE i INT
...
LET i = 1
DISPLAY i

=>> 1
```
