# SQLCA.SQLCODE

| SQLCODE 狀態 | 說明 |
| :--- | :--- |
| SQLCA.SQLCODE = 0 | 表示 SQL 執行成功 |
| SQLCA.SQLCODE = 100 | 表示 SQL 執行成功，但未有符合條件的資料被取出 |
| SQLCA.SQLCODE &lt; 0 | 表示 SQL 執行失敗 |

{% hint style="info" %}
#### 說明

反映 SQL 指令執行後的結果
{% endhint %}
