# SET\_COUNT( )

## 語法

```
SET_COUNT( intExpr )
```

{% hint style="info" %}
目的：

設定陣列的資料總筆數。
{% endhint %}

{% hint style="info" %}
說明：

expr ：算術表示式。

於 INPUT ARRAY WITHOUT DEFAULTS 敘述或 DISPLAY ARRAY 敘述前使用。
{% endhint %}
