# VARCHAR

#### 說明

| 語法   | VARCHAR ( n ) |
| ---- | ------------- |
| 資料長度 | n             |
| 資料範圍 | 1 <= n <= 255 |

#### 範例

```objectivec
DEFINE answer VARCHAR(1)
...
LET answer = "a"
DISPLAY answer

=>> a
```
