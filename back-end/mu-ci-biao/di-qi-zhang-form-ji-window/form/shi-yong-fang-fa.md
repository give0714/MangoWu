# 使用方法

## OPEN FORM

```inform7
OPEN FORM formName FROM "formFileName"
```

## DISPLAY FORM

```inform7
DISPLAY FORM formName [ Attribute ]
```

## CLOSE FORM

```inform7
CLOSE FORM
```

## CLEAR FORM

```
CLEAR FORM
```

{% hint style="info" %}
目的：

清除螢幕所有欄位資訊。
{% endhint %}
