# ERROR

## 語法1

```inform7
ERROR "str1" [, "str2", ... ] [ Attribute ]
```

## 語法2

```
ERROR variable1 [, variable2, ... ] [ Atrribute ]
```

{% hint style="info" %}
目的：

顯示告警訊息於 ERROR 行，會有嗶一聲。\
當按任意鍵時，會主動清楚螢幕的錯誤訊息行內資訊。\
預設位置在 SCREEN 最後一行。
{% endhint %}
