# Screen FORM 指定行

## 語法1

```inform7
DISPLAY ( "str1" [, "str2", ... ]
          / variable1 [, variable2, ... ]
          / TEXTvariable1 [, TEXTvariable2, ... ] )
        TO fieldName1 [, fieldName2, ... ] [ Atrribute ]
```

## 語法 2

```inform7
DISPLAY BY NAME variable1 [, variable2, ... ] [ Atrribute ]
```

## 語法 3

```inform7
DISPLAY BY NAME recordName.variableM THRU recordName.variableN [ Atrribute ]
```
