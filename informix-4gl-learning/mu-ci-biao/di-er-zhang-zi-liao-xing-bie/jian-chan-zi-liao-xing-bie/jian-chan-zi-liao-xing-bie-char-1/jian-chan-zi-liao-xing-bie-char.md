# CHAR

#### 說明

| 語法   | CHAR ( n )           |
| ---- | -------------------- |
| 資料長度 | n                    |
| 資料範圍 | 1 <= n <= 255 ( 位元 ) |

#### 範例

```objectivec
DEFINE answer CHAR(1)
...
LET answer = "a"
DISPLAY answer

=>> a
```
