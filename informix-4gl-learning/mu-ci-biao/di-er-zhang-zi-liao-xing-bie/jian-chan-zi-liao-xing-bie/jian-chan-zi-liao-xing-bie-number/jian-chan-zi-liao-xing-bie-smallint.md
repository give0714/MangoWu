# SMALLINT

#### 說明

| 語法   | SMALLINT        |
| ---- | --------------- |
| 資料長度 | 2 byte          |
| 資料範圍 | -32767 \~ 32767 |

#### 範例

```objectivec
DEFINE si SMALLINT
...
LET si = 1
DISPLAY si

=>> 1
```
