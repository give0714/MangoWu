# FLOAT

#### 說明

|  |  |
| :--- | :--- |
| 語法 | FLOAT |
| 資料長度 | 8 byte |
| 別名 | 倍精準浮點數 |
| 說明 | 相當為 C 語言的 DOUBLE 資料型別 |

#### 範例一

```objectivec
DEFINE f FLOAT
...
LET f = 11.0014
```

{% hint style="warning" %}
強烈建議將 FLOAT 資料型別改使用成 DECIMAL 資料型別
{% endhint %}
