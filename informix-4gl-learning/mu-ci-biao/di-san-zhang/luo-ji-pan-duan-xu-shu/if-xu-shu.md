# IF

```objectivec
IF booleanExpr THEN
   ...
[ ELSE 
       ... ]
END IF
```

{% hint style="warning" %}
備註

booleanExpr  結果只能為  TRUE  或  FALSE  ，不能為  UNKNOW
{% endhint %}
