# SQLCA.SQLERRD

| SQLERRD 狀態           | 說明                    |
| -------------------- | --------------------- |
| SQLCA.SQLERRD \[ 1 ] | Informix-4GL 保留字，尚未使用 |
| SQLCA.SQLERRD \[ 2 ] | 新增時， SERIAL 資料型別所傳回的值 |
| SQLCA.SQLERRD \[ 3 ] | 處理資料的筆數               |
| SQLCA.SQLERRD \[ 4 ] | 查詢時，預估的 CPU COST      |
| SQLCA.SQLERRD \[ 5 ] | SQL 指令的錯誤位移           |
| SQLCA.SQLERRD \[ 6 ] | 最後的 ROWID 值           |
