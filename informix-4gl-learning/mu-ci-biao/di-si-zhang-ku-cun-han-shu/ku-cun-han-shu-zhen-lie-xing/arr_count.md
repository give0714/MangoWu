# ARR\_COUNT( )

## 語法

```
ARR_COUNT( )
```

{% hint style="info" %}
目的：

傳回程式陣列的所有列總數量。
{% endhint %}

{% hint style="info" %}
說明：

用於傳回 INPUT ARRAY 敘述或使用者輸入的列數量。

可搭配 ARR\_COUNT( ) 和 SCR\_LINE( ) 使用。
{% endhint %}
