# ARR\_CURR( )

## 語法

```
ARR_CURR( )
```

{% hint style="info" %}
目的：

傳回程式陣列的目前列編號。
{% endhint %}

{% hint style="info" %}
說明：

用於傳回 INPUT ARRAY 敘述或 DISPLAY ARRAY 敘述目前列編號。

程式陣列和螢幕陣列的第一列編號為 1 。
{% endhint %}
