# ERR\_QUIT( )

## 語法

```
ERR_QUIT( intExpr )
```

{% hint style="info" %}
目的：

將錯誤訊息代碼，顯示於 WINDOWS 錯誤訊息行【ERROR Line】內。

intExpr：為負整數，通常為狀態變數 status，也可放置 SQLCA.SQLCODE，可提高偵錯效率，於執行完畢後，會停止程式運行。
{% endhint %}
