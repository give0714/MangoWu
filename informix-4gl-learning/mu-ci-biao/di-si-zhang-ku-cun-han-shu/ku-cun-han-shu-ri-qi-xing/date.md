# DATE( )

## 語法

```
DATE( expr )
```

{% hint style="info" %}
目的：

傳回 DATE 資料型態的資料，只接受 YEAR、MONTH、DAY 的資料。
{% endhint %}

{% hint style="info" %}
說明：

expr ：可被轉換成 DATE 資料型態的表示式。
{% endhint %}
