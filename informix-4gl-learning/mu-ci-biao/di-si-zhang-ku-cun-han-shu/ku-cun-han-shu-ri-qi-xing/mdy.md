# MDY( )

## 語法

```
MDY( expr1, expr2, expr3 )
```

{% hint style="info" %}
目的：

輸入三個整數值，並將其轉乘時間型別。
{% endhint %}

{% hint style="info" %}
說明：

expr1 ：代表月份 ( 1 \~ 12 ) 的整數值。

expr2 ：代表日期 ( 1 \~ 31 ) 的整數值。

expr3 ：代表年份 ( 0000 \~ 9999 ) 的整數值。
{% endhint %}
