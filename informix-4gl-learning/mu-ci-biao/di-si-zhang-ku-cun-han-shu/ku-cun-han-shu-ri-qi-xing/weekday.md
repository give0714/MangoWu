# WEEKDAY( )

## 語法

```
WEEKDAY( expr )
```

{% hint style="info" %}
目的：

輸入一時間型別的資料，將此日期換算成對應星期，並傳回一代表星期的整數值。
{% endhint %}

{% hint style="info" %}
說明：

範圍 0 \~ 6，0 為星期日，1為星期一，以此類推。
{% endhint %}
